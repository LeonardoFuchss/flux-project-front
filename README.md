# Flux – Frontend Web Application

## 📌 Description
This repository contains the **frontend application** for **Flux – Personal Finance Control System**.

The frontend is responsible for providing the user interface that allows users to interact with the Flux backend API, enabling personal finance management through an intuitive and responsive web experience.

🔗 **Backend repository:**  
https://github.com/LeonardoFuchss/flux-project

---

## 🚀 MVP Features
- User registration and authentication
- Income and expense management
- Transaction listing
- Transaction filtering
- Category-based visualization
- Monthly balance overview

---

## 🔄 User Flow Overview
1. User registers or logs into the application  
2. Authentication token is generated by the backend  
3. Frontend stores and sends the token on protected requests  
4. User creates, updates, and deletes transactions  
5. Transactions are organized by category  
6. Monthly balance is calculated and displayed
